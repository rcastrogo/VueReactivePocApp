FORM SERIALIZER LAB

Prueba visual de serializacion de formularios

Escenario manual para inspeccionar como se serializan distintos tipos de campos usando data-field.

Incluye inputs de texto, selects, textarea, radios, checkboxes, campos deshabilitados, paths tipo address.city y multiples checks con el mismo name.

OBSERVACIONES

  • Los campos deshabilitados siguen entrando en la serializacion actual.
  • Si varios checkboxes comparten name pero tienen distinto data-field, se serializan por separado.
  • Si varios checkboxes comparten el mismo data-field, se agregan en un array de valores.
  • Los data-field con rutas generan objetos anidados y se pueden hidratar con apply.
